### 1.2

**Solution**

```python
# Single lines:

print("*'-.-'*'-.-'*'-.-'*'--'*")
print("    Project: Python")
print("     Version: 1.0")
print("Author: Guido van Rossum")
print(" Email: bdfl@python.com")
print("    Date: Jan, 1994")
print("*'-.-'*'-.-'*'-.-'*'--'*")


# Multi-line

print('''
    *'-.-'*'-.-'*'-.-'*'--'*
        Project: Python
        Version: 1.0
    Author: Guido van Rossum
    Email: bdfl@python.com
        Date: Jan, 1994
    *'-.-'*'-.-'*'-.-'*'--'*
    '''
) # notice how this parenthesis
#is at the same indentation as print()
```
