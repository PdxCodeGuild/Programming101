### <a id="strings"></a>String Concatenation
To concatenate, or combine, two strings you can use the + operator.

Example 1:
```python
a = "Hello"
b = "World"
c = a + b
print(c) # outcome: "HelloWorld"
```

Example 2:
```python
a = "Hello"
b = "World"
c = a + " " + b
print(c) # outcome: "Hello World"
```
